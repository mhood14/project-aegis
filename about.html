About This Project

Why this project was built.

Project Aegis was built to learn how to stand up a fully remote Microsoft tenant, secure network infrastructure, and an AI application that can retrieve internal documents with hybrid search and logging.

01

Project goal

The goal was to learn how to set up a fully remote Microsoft tenant, network infrastructure, and AI application in one cohesive environment instead of treating each skill as a separate exercise.

02

How the learning happened

The project became a hands-on way to learn Microsoft tenant setup, secure configuration, Sentinel setup and security monitoring, secure networking, hybrid search, and practical AI integration inside that environment.

03

What was built

To make that learning concrete, Project Aegis implements a secure document retrieval app with scoped access, hybrid search, grounded responses, and logging that follows activity across the system.

04

Why it matters

The project is both a way to learn these skills and a portfolio piece showing what was learned through an end-to-end Microsoft cloud, networking, security, and AI build.

🎯

Career direction

The long-term goal is to get a job with Microsoft and/or do contracting work helping companies build secure remote environments or securely implement AI.

🧠

What the site is meant to show

This site presents the project as a standalone system: the architecture, the live app, the logging and testing story, and the supporting artifacts that make the build credible.
